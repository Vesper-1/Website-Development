Introduction

HTML (HyperText Markup Language) defines the structure of web pages using elements represented by tags.

Document structure

Basic HTML document skeleton:

<!doctype html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width,initial-scale=1">
        <title>Page title</title>
    </head>
    <body>
        <!-- content -->
    </body>
</html>

Common elements

Examples of frequently used elements:

<h1>Heading</h1>
<p>Paragraph text with <a href="#">link</a>.</p>
<img src="image.jpg" alt="description">
<ul><li>Item</li></ul>

Attributes

Attributes provide additional information for elements. Common attributes: id, class, src, href, alt, title, data-*

<button id="saveBtn" class="primary" data-action="save">Save</button>

Forms & inputs

Form basics with labels and input types:

<form action="/submit" method="post">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <button type="submit">Send</button>
</form>

Semantic HTML

Use semantic tags for better structure and accessibility: <header>, <nav>, <main>, <article>, <section>, <footer>.

Accessibility

Basics: provide alt text, associate labels with inputs, use meaningful semantics, ensure keyboard focus, and use ARIA when necessary.

Resources

MDN Web Docs and WHATWG HTML Living Standard are authoritative references.
